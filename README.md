# JibCheck-Ver1
This Git repository contains a PowerShell-based Network Performance Testing Tool. This tool allows users to assess network performance by conducting network speed tests between a server and client. It offers a menu-driven interface for selecting options, including starting a server, initiating a client test, and exiting the application.

Usage:
Run the server script (listening server).

Run the app in claent mechene.
Specify the serial number and other details when prompted.
The tool will perform the test and generate logs.
Test results are evaluated against criteria from a JSON file.
Results are logged, indicating whether the test passed or failed.
